## üöÄ Quick Start

### For Business Users
1. Visit the live dashboard at the deployment URL
2. Browse ETFs using search or category filters
3. Select time periods to analyze performance trends
4. Click on ETFs to view detailed composition and metrics

### For Developers
1. Clone this repository
2. Install dependencies: `npm install`
3. Set up Alpha Vantage API key in environment variables
4. Run development server: `npm run dev`
5. Access API testing console at `/api-test`
